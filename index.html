Kelas Mobil - Named Constructor

Memiliki parameterized constructor, named constructor baru() dan dariJson(). FLEXIBLE LEVEL: AMAN

mobil.dart

DART SECURE 
// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized Constructor
  Mobil(this.merk, this.tahun);

  // Named Constructor: Mobil Baru
  Mobil.baru() {
    merk = 'Cyber Truck';
    tahun = 2025;
  }

  // Named Constructor: dari JSON
  Mobil.dariJson(Map<String, dynamic> json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }
}

Kelas User - Login System

Enkapsulasi dengan metode login dan ganti password. AUTHENTICATION LEVEL: KRITIS

user.dart

SECURE AUTH 
class User {
  String _username;
  String _password;
  String _email;

  User(this._username, this._password, this._email);

  bool login(String username, String password) {
    return username == _username && password == _password;
  }

  void gantiPassword(String lama, String baru) {
    if (lama == _password && baru.length >= 6) {
      _password = baru;
    }
  }
}
